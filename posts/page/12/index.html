Ruby present? Method

Ruby has a nice little keyword called unless, that checks the opposite of if. So, you are probably used to a code like this: return customer.first_name unless customer.nil? If you haven't used present? before, you can in fact turn the above unless into a more familiar and easy to understand if statement: return customer.first_name if customer.present? So, in most cases when you are using unless with a negative condition, you can use present?
