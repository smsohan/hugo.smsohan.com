What's wrong with the default alert rendered by the browser?

MicroOptimization Trap

In short, given the following code in CoffeeScript: class Cat type: -> 'cat' meow: -> 'meow meow mee..ow' The compiler produces this in JavaScript: var Cat; Cat = (function() { function Cat() {} Cat.prototype.type = function() { return 'cat'; }; Cat.prototype.meow = function() { return 'meow meow mee..ow'; }; return Cat; })(); As you see here, the prototype method definitions use the prefix Cat.
